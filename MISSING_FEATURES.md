## 🚧 빠진 기능 (우선순위별 정리)

### 🔥 높은 우선순위 (즉시 구현 필요)

#### 1. API의 PUT/DELETE 메서드 미구현

현재 대부분의 API가 GET과 POST만 구현되어 있습니다. 수정(PUT)과 삭제(DELETE) 기능이 필요합니다.

##### 수익 관리 API (`/api/revenue`)
- ❌ PUT: 수익 데이터 수정
- ❌ DELETE: 수익 데이터 삭제
- **현재 상태**: GET, POST만 구현됨

##### 지출 관리 API (`/api/expenses`)
- ❌ PUT: 지출 데이터 수정
- ❌ DELETE: 지출 데이터 삭제
- **현재 상태**: GET, POST만 구현됨

##### 캘린더 이벤트 API (`/api/calendar`)
- ❌ PUT: 캘린더 이벤트 수정
- ❌ DELETE: 캘린더 이벤트 삭제
- **현재 상태**: GET, POST만 구현됨

##### 코멘트 API (`/api/comments`)
- ❌ PUT: 코멘트 수정
- ❌ DELETE: 코멘트 삭제
- **현재 상태**: GET, POST만 구현됨

##### 피드백 API (`/api/feedback`)
- ❌ PUT: 피드백 수정
- ❌ DELETE: 피드백 삭제
- **현재 상태**: GET, POST만 구현됨

##### 알림 API (`/api/notifications`)
- ❌ DELETE: 알림 삭제
- **현재 상태**: GET, POST, PUT만 구현됨

##### 사용자 관리 API (`/api/users`)
- ❌ DELETE: 사용자 삭제
- **현재 상태**: GET, POST, PUT만 구현됨

#### 2. 수정/삭제 UI 페이지 미구현

##### 수익 관리
- ❌ 수익 수정 페이지 (`/revenue/[id]/edit`)
- ❌ 수익 목록에서 수정/삭제 버튼 및 기능

##### 지출 관리
- ❌ 지출 수정 페이지 (`/expenses/[id]/edit`)
- ❌ 지출 목록에서 수정/삭제 버튼 및 기능

##### 캘린더
- ❌ 캘린더 이벤트 수정 기능
- ❌ 캘린더 이벤트 삭제 기능
- ❌ 캘린더 이벤트 추가 페이지 (`/calendar/new`)

##### 코멘트
- ❌ 코멘트 수정 기능
- ❌ 코멘트 삭제 기능

##### 피드백
- ❌ 피드백 수정 기능
- ❌ 피드백 삭제 기능
- ❌ 피드백 답변 기능 (현재 버튼만 있음)

#### 3. 실제 파일 스토리지 연동

##### 음원 파일 업로드
- ❌ Cloudflare R2 또는 실제 스토리지 연동
- **현재 상태**: 시뮬레이션된 URL만 반환
- ❌ 파일 메타데이터 추출
- ❌ 파일 미리듣기 기능
