# SUNO 새 프로젝트 페이지 – 추가 보완할 부분 (요약)

## 이미 반영된 것
- Styles/Exclude maxlength·검증, 커버 인라인 에러
- 아코디언 aria-expanded/aria-controls, transition:slide
- handleSubmit 검증·에러 처리, 폼 이탈 확인
- 드롭다운 aria, 소속 앨범(목록+직접 입력), 뮤직비디오 제작·연결
- 체크박스 대시보드 스타일, 활성 시 호버 색 유지

---

## 나중에 하면 좋은 것 (우선순위)

| 순위 | 항목 | 내용 |
|------|------|------|
| 1 | **포커스 링 전환** | 아코디언 제목 Tab 포커스 시 링이 0.15초 정도 서서히 나타나게 (`transition: box-shadow`) |
| 2 | **Simple 모드 안내** | "프롬프트 · 가사"에서 Simple일 때 "스타일만 입력하는 모드예요" 한 줄 안내 |
| 3 | **모바일 미리보기** | 화면 좁을 때 sticky 해제 또는 "미리보기 접기" 버튼 |
| 4 | **필드별 에러** | 제목/Styles 길이 등 검증 실패 시 해당 필드 옆에 빨간 메시지 표시 |
| 5 | **모바일 버튼** | "취소" / "프로젝트 생성"을 매우 좁을 때 세로 배치 또는 풀폭 |
